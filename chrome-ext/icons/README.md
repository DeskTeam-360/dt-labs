# Icons

Tempatkan icon extension di sini dengan ukuran:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)  
- icon128.png (128x128 pixels)

Anda bisa membuat icon menggunakan tool online seperti:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/

Atau gunakan placeholder icon untuk testing.
